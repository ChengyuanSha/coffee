# Mulab Coffee Project 

Analyze coffee consumption behaviour and prefereces

## Usage
Run in terminal:<br />
npm start <br />
Button config file: /src/config.json <br />
Compile new button: /src/gen_files.py <br />
Combine output csv: /src/combine_output.py <br />

## Requirements
python <br />
nodejs > 1.4 <br />
npm <br /> 
pandas <br />
